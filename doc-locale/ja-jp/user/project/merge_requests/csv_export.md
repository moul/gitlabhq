プロジェクトのマージリクエストから収集されたすべてのデータを、カンマ区切り値（CSV）ファイルにエクスポートします。

マージリクエストをCSVファイルにエクスポートするには:

1. 上部のバーで、**検索または移動先**を選択して、プロジェクトを見つけます。
1. 左サイドバーで、**コード** > **マージリクエスト**を選択します。
1. 検索またはフィルターを追加します。これにより、CSVファイルのサイズを15 MBの制限内に保つことができます。この制限により、ファイルをさまざまなメールプロバイダーにメールで送信できます。
1. **アクション** ({{< icon name="ellipsis_v" >}}) > **CSV形式でエクスポート**を選択します。
1. マージリクエストの正しい数がエクスポートされることを確認します。
1. **マージリクエストをエクスポート**を選択します。

## CSV出力 {#csv-output}

次の表は、CSVファイルの属性を示しています。

| カラム             | 説明                                                  |
|--------------------|--------------------------------------------------------------|
| タイトル              | マージリクエストのタイトル                                          |
| 説明        | マージリクエストの説明                                    |
| MR ID              | MR `iid`                                                     |
| URL                | GitLab上のマージリクエストへのリンク                        |
| ステート              | オープン、クローズ、ロック、またはマージ済み                            |
| ソースブランチ      | ソースブランチ                                                |
| ターゲットブランチ      | ターゲットブランチ                                                |
| ソースプロジェクトID  | ソースプロジェクトのID                                     |
| ターゲットプロジェクトID  | ターゲットプロジェクトのID                                     |
| 作成者             | マージリクエストの作成者のフルネーム                        |
| 作成者ユーザー名    | @シンボルが省略された、作成者のユーザー名            |
| 担当者          | マージリクエストの担当者のフルネーム（`,`で結合） |
| 担当者ユーザー名 | @シンボルが省略された、担当者のユーザー名         |
| 承認者          | 承認者のフルネーム（`,`で結合）               |
| 承認者ユーザー名 | @シンボルが省略された、承認者のユーザー名         |
| マージ済みユーザー        | マージ済みユーザーのフルネーム                                 |
| マージ済みユーザー名    | @シンボルが省略された、マージユーザーのユーザー名        |
| マイルストーンID       | マージリクエストのマイルストーンID                            |
| 作成日時（UTC）   | `YYYY-MM-DD HH:MM:SS`形式                           |
| 更新日時（UTC）   | `YYYY-MM-DD HH:MM:SS`形式                           |

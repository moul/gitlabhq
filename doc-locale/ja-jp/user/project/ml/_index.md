GitLab機械学習Operations (MLOps) は、機械学習ワークフローを支援するために設計されたツールセットです。

GitLab MLOpsの機能は以下のとおりです:

- モデルレジストリ: MLモデル、およびパラメータ、パフォーマンスメトリクス、アーティファクト、ログなどの関連メタデータを管理します。詳細については、[モデルレジストリ](model_registry/_index.md)を参照してください。
- モデル実験: GitLabで機械学習の実験を追跡および管理します。実験とは、比較可能なMLモデル候補のコレクションであり、MLモデルのトレーニングのバリエーションです。詳細については、[モデル実験](experiment_tracking/_index.md)を参照してください。

## GitLab MLOps Pythonクライアント {#gitlab-mlops-python-client}

GitLabは、GitLab MLOps機能と対話するためのPythonクライアントを提供します。

詳細については、[GitLab MLOps Pythonクライアント](https://gitlab.com/gitlab-org/modelops/mlops/gitlab-mlops)を参照してください。
